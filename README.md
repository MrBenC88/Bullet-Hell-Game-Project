# Project-Twilight
A top-view shooting game that is based off of Touhou.
Programmed in Java.

Features:
- In Game Store and Currency System
- Adjustable Levels of Difficulty
- Custom Roster of Playable Characters
- Multiple unique attack patterns
- Uses Java Swing
